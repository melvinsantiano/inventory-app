# Inventory App

A simple inventory management application for tracking products, stock levels, and inventory operations.

## Features

- **Product Management**: Add, edit, and delete products
- **Stock Tracking**: Monitor inventory levels
- **Search & Filter**: Find products quickly
- **Data Persistence**: Local storage for data
- **Responsive Design**: Works on desktop and mobile devices

## Files

- `index.html` - Main application interface
- `script.js` - Application logic
- `style.css` - Styling and layout
- `mds-logo.png` - Application logo

## Usage

Simply open `index.html` in a web browser to run the application.

## Live Demo

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start managing your inventory!
